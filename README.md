# TMDB Movie Browser with JWT Authentication

A React web application to browse and search movies using The Movie Database (TMDB) API.  
Supports user authentication with JWT tokens and pagination of movie results.

---

## Features

- Browse popular movies from TMDB  
- Search movies by keyword  
- Pagination controls for easy navigation  
- User login and registration with JWT authentication  
- JWT stored in localStorage and sent with API requests  
- Movie details including poster, title, overview, and release date
